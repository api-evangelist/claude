Anthropic is broadening its Project Glasswing initiative, which aims to strengthen software security globally. The program initially engaged roughly 50 partners who deployed Claude Mythos Preview to identify code vulnerabilities, discovering more than 10,000 high- or critical-severity security flaws in their systems. The expansion now includes approximately 150 additional organizations spanning over 15 countries and multiple critical sectors including power, water, healthcare, communications, and hardware.
